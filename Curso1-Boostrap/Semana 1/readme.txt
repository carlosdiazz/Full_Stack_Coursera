Aqui en esta semanas noes enfocamos en:

Objetivos de aprendizaje de la Semana 1:

    Expresar lo que se entiende por pila completa en el contexto del desarrollo web
    
    Cree un proyecto con soporte Bootstrap
    
    Cree un sitio web receptivo utilizando el sistema de cuadrícula Bootstrap
    
    Desarrolle un sitio web que pueda adaptarse automáticamente a diferentes tamaños y resoluciones de pantalla

Objetivos de aprendizaje de la Semana 2:

    Exprese la necesidad de incluir la navegación en su sitio web
    
    Cree funciones de navegación en su sitio web
    
    Cree presentaciones de datos utilizando tablas y tarjetas
   
    Prepare los medios para incluirlos en su sitio web
    
    Cree medios para proporcionar interacción del usuario con su sitio web mediante formularios y botones

Objetivos de aprendizaje de la Semana 3


    Reconozca cómo los componentes de JavaScript proporcionan un comportamiento dinámico a su sitio web.

    Cree varios elementos de navegación en su sitio web usando Tabs y Accordion

    Cree modales, información sobre herramientas y ventanas emergentes para revelar contenido en su sitio web

    Cree un carrusel para mostrar contenido dinámico en su sitio web
